# Tetris.cpp

Simple Implementation of Tetris Game in C++.

![Demo](./demo.png)

## How to play

Supports both Linux and MacOS.

```bash
./build.sh
```

By default, the executable is `./build/bin/SimpleTetris`.
The game will run in a 42x22 terminal window.

## Controls

- `h`: move left
- `l`: move right
- `j`: move down
- `k`: rotate
- `p`: pause
- `q`: quit
